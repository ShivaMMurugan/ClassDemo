# SSH - SECURE SHELL HASHING Or SECURE SHELL ENCRYPTION
```sh
Type ssh-keygen
The above command will create a directory called .ssh
under .ssh / cd .ssh
You can find two files one is id_rsa.pub and the other one is id_rsa.private
The id_rsa.pub is known as a public key which used to authenticate yourself
to third party applications or softwares.
You have to copy the id_rsa.pub values to github. This will set the RSA Encryption keys to 
your machine.
```
